Configuration File
snake_10_GJ.conf - a snake shaped configuration

Gait Files
forward.gait - move the configuration forward for a short distance
forward2.gait - move the configuration forward for a long distance
turnleft.gait - turn the configuration left
turnright.gait - turn the configuration right
bendup2.gait - lift the snake head up
bendup.gait - climb table

Control Example
Use forward.gait to reach to any object for picking up.
Use bendup2.gait to lift the object off the ground.
Use forward2.gait to reach to the table
Use bendup.gait to climb the table
